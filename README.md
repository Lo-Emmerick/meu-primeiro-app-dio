# Meu Primeiro App – Bootcamp Mobile Developer (DIO)
Este projeto foi criado durante o desafio do Bootcamp da DIO para praticar os conceitos básicos de desenvolvimento Android com Kotlin.

O aplicativo possui uma interface simples, onde o usuário pode escolher um idioma e ver o conteúdo mudar automaticamente.

# Tecnologias Utilizadas

* Kotlin
* Android Studio
* ViewBinding

# Funcionalidades

* Escolha de idioma em uma lista.
* Atualização dos textos conforme o idioma selecionado.
